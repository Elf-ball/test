# 内网穿透

- 利用反向代理的原理，花生壳的内网穿透服务就是新开了一个反向代理服务，该代理与本地电脑的上web服务属于同一局域网，该代理又可以被公网上的客户端访问。配置代理时一般都会自己定义一个代理的域名，外网通过访问该域名访问到内网服务器
